## About

Bloom is a mobile flower shop application designed with a focus on HCI principles. It allows users to browse, customize, and order flower arrangements for any occasion — birthdays, weddings, condolences, and more — with a smooth, user-centered shopping experience.

## Screens Included

- **Splash Screen** — App branding and loading
- **Login / Signup** — User authentication screens
- **Home Screen** — Featured arrangements, categories, promotions
- **Browse / Catalogue** — Filter and explore flower collections
- **Product Detail** — Item info, customization, add to cart
- **Cart & Checkout** — Order review and payment
- **Order Tracking** — Live order status updates
- **Profile** — User info, order history, preferences

---

## HCI Principles Applied

- **Usability** — Simple navigation, minimal steps to complete an order
- **Consistency & Standards** — Uniform color palette, typography, and button styles throughout
- **User-Centered Design** — Flows designed around real user goals (browse → customize → order)
- **Feedback & System Response** — Button states, loading indicators, order confirmations
- **Error Prevention** — Input validation on forms, confirmation before placing orders
- **Accessibility** — Sufficient color contrast, readable font sizes, clear iconography
- **Visual Hierarchy** — Clear layout structure guiding the eye from key actions to supporting info
